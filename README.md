## Hey there,

This is Vanilla JavaScript + Tailwind CSS template.

Use this template via `@pr4j3sh/frames`:

```bash
npm create @pr4j3sh/frames js-tw myapp
```

> You can replace `myapp` with the choice of your name.

Run the app using:

```bash
npm run dev
```
